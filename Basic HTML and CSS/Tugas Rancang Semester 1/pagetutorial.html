Sistem Operasi

Sistem Operasi adalah Software layer pertama yang menjembatani dan memanage seluruh hardware dan software yang terdapat di komputer. Sistem Operasi juga menyertakan banyak software di dalamnya seperti System Services, libraries, dan API (Application Programming Interfaces) yang dimana developer bisa menulis programnya di atas Sistem Operasi.

Sistem operasi bekerja dan berjalan di antara hardware dan program apapun yang berjalan di atas sistem operasi mobile ataupun komputer. Beberapa peran oleh sistem operasi yakni membantu managing input yang dimasukan oleh users, mengirimkan output kepada output devices, manage storage spaces, dan mengontrol seluruh input output devices peripheral terhubung.

Sejak perilisan perdananya, iOS telah dipakai untuk sistem operasi perangkat yang dikembangkan oleh Apple Incorporation, yaitu iPhone, iPad, iPod Touch, dan Homepad. iOS memperkenalkan konsep Multitouch dan User Interface untuk pertama kali yang fresh bagi user pada jamannya.

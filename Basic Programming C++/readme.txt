Basic Pemrograman :
- Variabel dan Deklarasi 
- Tipe data (int, float, string, char, double, bool)
- Aritmatika (+, -, /, *, %)
- Operator logika (OR, AND, NOT)
- cout dan cin
- Increment dan Decrement (++, +=, --, -=)
- Percabangan (if, else if, else) (switch case)
- Perulangan (while, do while, for)
- Fungsi (return, void)
